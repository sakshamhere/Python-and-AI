
1. `Transformers Library`: includes thousands of pre-trained models, which can be fine tuned for specific tasks.

- Open source python library that provides NLP models and tools, it is built arounf transformer architecture, it inludes wide range of pre-trained models for tasks such as text classification, translation, summarization, chat etc

2. `Model Hub`: a place where users can share pre-trained models, datasets, etc.

3. `Datasets library`: Provides easy access to wide variety of datasets for NLP tasks.

4. `Spaces`: A platform for hosting and sharing machine learning demos and applications.

5. `Inferece API`: Allow users to easily deploy and use models in production enviornment


- `Tokenizers Library`: fast, efficient and fast library designed for tokenizing text data
